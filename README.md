## How to Install   
First, install dependencies   
```bash
# clone project   
git clone https://github.com/mbencherif/capt-listen

# install project   
cd capt-listen
pip install -e .   
pip install -r requirements.txt
 ```   
## How to run   
```bash
cd capt-listen
streamlit run main.py --server.port 8502
```
